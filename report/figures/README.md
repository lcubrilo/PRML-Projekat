# Figures

All plots used in the report go here, one PNG per figure. Generate them with
`src.plotting.save_fig("name")` so they all use the same path and resolution.
The report (`report/report.md`) references them as `figures/name.png`.

Suggested names (so the report can link them before they exist):
`class_balance`, `method_distribution`, `red_corner_confound`, `feature_correlations`,
`confusion_matrix_<model>`, `hyperparam_k_knn`, `hyperparam_samme`, `pca_2d`, `model_comparison`.
